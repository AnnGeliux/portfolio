## Resumen

Sistema web para gestionar lecturas: registro de libros, definición de metas
y seguimiento del progreso de lectura. Pensado para clubes de lectura,
bibliotecas personales y entornos educativos.

## Características

- Registro y catálogo de libros con metas de lectura.
- Panel de estadísticas con el avance por usuario.
- Control de usuarios y permisos.
- Interfaz moderna construida con HTML, CSS y JavaScript.
